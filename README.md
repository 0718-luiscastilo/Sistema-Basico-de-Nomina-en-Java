 Sistema Básico de Nómina en Java

Este proyecto es una aplicación de consola desarrollada en Java que permite calcular el pago de empleados en función de sus horas trabajadas, incluyendo horas extra.

Funcionalidades

- Registro de múltiples empleados
- Cálculo de pago por horas trabajadas
- Aplicación automática de horas extra (más de 8 horas)
- Acumulación de:
  - Total de empleados procesados
  - Suma total de pagos
- Interacción mediante consola

Conceptos aplicados

- Variables y tipos de datos
- Operadores aritméticos
- Estructuras de decisión (`if`)
- Bucles (`do-while`)
- Funciones (métodos)
- Entrada de datos con `Scanner`

Lógica del sistema

- Las primeras 8 horas se pagan normalmente
- Las horas adicionales se pagan al doble
- El sistema permite ingresar múltiples empleados hasta que el usuario decida salir

Tecnologías utilizadas

- Java (JDK estándar)
- Consola (CLI)
